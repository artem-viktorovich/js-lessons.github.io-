\n переноса строки с кодом на другую строку
Но также перенос можно осуществлять с помощью модульных кавычек

let a = 7;
console.log(`Text 
on 2 line ${a}`);


При использовании обычный кавычек, надо плюсовать переменные

  let a = 12;

Math.max(3,9,7,5,10,7) - возврат максимального значения с перечисленных чисел (либо иных объектов)
10


Math.min(3,9,7,5,10,7) - возврат минимального значения с перечисленных чисел (либо иных объектов)
3


Math.pow(2,5) - возведение числа в степень
32


Math.pow(16,0.5) - 0,5 это число для выведения квадратного корня с числа
4

Math.sqrt(16) - выведение корня с целого числа


Math.floor(3.59590) - метод для округления числа в меньшую сторону
3

Math.ceil(3.59590) - округляет число в большую сторону

Math.round(3.5) - округляет число к ближайшему целому
4

Math.trunc(3.1) - отбрасывание дробной части
3
